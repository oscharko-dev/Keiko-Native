# Slint candidate licence record

This throwaway desktop prototype pins Slint `1.17.1` and selects
`LicenseRef-Slint-Royalty-free-2.0` from Slint's declared licence expression.
The UI includes the official `AboutSlint` attribution widget in its main window,
satisfying attribution option 2(a) for this packaged experiment.

- [authoritative terms](https://slint.dev/terms-and-conditions#royalty-free)
- [stable licence PDF](https://slint.dev/agreements/slint-royalty-free-license.pdf)
- [crate metadata](https://crates.io/crates/slint/1.17.1)

`slint`, `slint-build`, and `rfd` are exact-version dependencies. The workspace
lockfile and generated SBOM remain the authoritative transitive inventory. No
paid Slint GUI automation, support service, live preview, interpreter, or remote
viewer is used.
